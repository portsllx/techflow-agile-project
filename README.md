# Construção de Projeto Ágil no GitHub: Da Gestão ao Controle de Qualidade

**Objetivo:** Simular o desenvolvimento de um sistema de gerenciamento de tarefas aplicando metodologias ágeis e boas práticas de Engenharia de Software.

# Sistema de Gerenciamento de Tarefas

Aplicação Flask simples para gerenciar as tarefas.

## Funcionalidades
- CRUD de tarefas
- Interface web básica
- Testes automatizados

## Tecnologias
- Python 3.x
- Flask
- Pytest

Filtro por Status de Tarefas
O que foi adicionado:

Filtro visual na interface web com três opções:

Pending (Pendentes): Exibe apenas tarefas não concluídas
Done (Concluídas): Exibe apenas tarefas finalizadas
